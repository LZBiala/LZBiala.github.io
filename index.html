U.S. Tornado Data Project

Aggregated Tornado data in the United States from 1950 to 2018. A Cost impact of tornado based on tornado magnitude interactive D3 bubble chart and Plotly graphs displaying some inferences and trends. Used a Leaflet interactive map displaying year and state with descriptions of the magnitude of the tornado and the distance.

ML Breast Cancer Data Project

Implemented Machine Learning Regressions to predict if a person's breast cancer tumor is malignant or benign. After we compared the accuracy of the three classifiers and found some good inferences from these models. Models used were:

* Logistic Regression Model
* Support Vector Machine Model
